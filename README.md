# MiniVerbe

Entraînement à la **conjugaison française** pour enfants. Interface et UX inspirées de MiniMath.

## Démarrage
```bash
npm install
npm start
```

## Build
```bash
npm run build
```

## Déploiement GitHub Pages
```bash
npm run deploy
```

## Docker
```bash
just start
# puis http://localhost:9999
```
